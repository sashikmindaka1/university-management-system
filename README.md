# 🎓 University ERP System

A modern **University Enterprise Resource Planning (ERP) System** developed to simplify and automate academic and administrative processes within a university. This project integrates student management, course registration, attendance tracking, and examination management into one centralized platform.

---

## 📖 Project Overview

The University ERP System helps educational institutions manage their daily operations efficiently by reducing manual work, improving data accuracy, and providing real-time access to information.

The system includes secure authentication, role-based access control, and cloud-based deployment using Microsoft Azure.
